# LLM Learning Assistant 📱

## About the App
This is an Android app I built to help students learn in a simple and interactive way.  
Instead of just giving answers, the app helps users understand concepts better using hints, explanations, flashcards, and a basic study plan.

---

## What the App Can Do

- Generate a learning task based on selected interests  
- Give hints to help solve questions  
- Explain why an answer is correct or incorrect  
- Create flashcards for quick revision  
- Suggest a simple 7-day study plan  
- Show both the prompt and the response clearly  
- Handle loading and error messages properly  
- Provide a clean and easy-to-use interface  

---

## Screens in the App

- Login and Register  
- Interests selection  
- Home screen  
- Task screen  
- Results screen  
- Flashcards screen  
- Study plan screen  

---

## Tools Used

- Java  
- XML

## How to Run

1. Open the project in Android Studio  
2. Let Gradle sync  
3. Run the app on an emulator or phone
